# ESP8266 WiFi破解工具

一个基于ESP8266的WiFi密码破解工具，具有Web界面配置、密码字典管理和飞书通知功能。

## 功能特性

- 🔍 **WiFi网络扫描**: 自动扫描附近所有WiFi网络
- 🔑 **密码字典破解**: 使用自定义密码字典进行暴力破解
- 🌐 **Web界面管理**: 通过浏览器进行配置和监控
- 📱 **飞书通知**: 破解成功后自动发送飞书消息
- 💾 **配置持久化**: 使用EEPROM保存配置信息
- 🎯 **目标选择**: 可选择特定WiFi或全部网络进行破解
- 💡 **LED状态指示**: 通过板载LED显示破解状态

## 硬件要求

- ESP8266开发板 (NodeMCU, Wemos D1 Mini等)
- 支持WiFi STA+AP模式
- 板载LED指示灯 (GPIO2/D4)

## 软件依赖

项目使用以下Arduino库：
- `ESP8266WiFi`
- `ESP8266WebServer`
- `ESP8266HTTPClient`
- `WiFiClientSecure`
- `EEPROM`

## 快速开始

### 1. 硬件连接

- 将ESP8266开发板连接到电脑
- 确保板载LED正常工作 (GPIO2/D4)

### 2. 代码上传

1. 使用Arduino IDE打开 `ESP8266_WIFI_CRACK.ino`
2. 选择正确的开发板和端口
3. 设置Flash Size为至少1MB
4. 上传代码到ESP8266

### 3. 初始连接

1. 打开串口监视器 (波特率115200)
2. 在WiFi列表中找到名为 `WiFi_CRACK` 的网络
3. 连接密码: `P0ssw0rd`
4. 浏览器访问: `http://192.168.4.1`

## 配置说明

### 默认配置

- **AP名称**: `WiFi_CRACK`
- **AP密码**: `P0ssw0rd`
- **管理地址**: `192.168.4.1`
- **子网掩码**: `255.255.255.0`
- **网关**: `192.168.4.1`
- **DHCP范围**: `192.168.4.2 - 192.168.4.254`

### Web界面功能

#### 主页 (`/`)
- 系统状态概览
- 快速访问各个功能模块

#### AP配置管理 (`/ap-config`)
- 修改AP名称和密码
- 配置保存后自动重启AP

#### 密码字典配置 (`/password`)
- 管理密码字典列表
- 支持逗号或换行分隔
- 默认包含常用弱密码

#### WiFi扫描 (`/wifi-scan`)
- 扫描附近所有WiFi网络
- 显示信号强度、加密方式等信息

#### 目标WiFi设置 (`/target-wifi`)
- 选择特定WiFi进行破解
- 或选择破解全部发现的网络

#### 飞书通知配置 (`/webhook`)
- 配置飞书机器人Webhook URL
- 破解成功后自动发送通知

#### 开始破解 (`/start-crack`)
- 启动WiFi密码破解程序
- 后台运行，可通过串口监控进度

## 飞书通知配置

### 1. 创建飞书机器人

1. 打开飞书，进入需要接收通知的群聊
2. 点击群设置 → 群机器人 → 添加机器人
3. 选择"自定义机器人"
4. 设置机器人名称和描述
5. 复制生成的Webhook URL

### 2. 配置Webhook

1. 访问Web管理界面: `http://192.168.4.1/webhook`
2. 粘贴飞书Webhook URL
3. 点击保存配置

### 3. 通知格式

破解成功后，飞书将收到消息。

## EEPROM存储结构

| 配置项 | 地址 | 最大长度 | 描述 |
|--------|------|----------|------|
| Webhook URL | 0 | 255 | 飞书Webhook地址 |
| 密码字典 | 512 | 1023 | 密码字典内容 |
| AP名称 | 1536 | 31 | AP的SSID名称 |
| AP密码 | 1792 | 63 | AP的连接密码 |
| 目标WiFi | 2048 | 31 | 指定的目标WiFi名称 |

## 技术细节

### 破解算法

1. **网络扫描**: 使用STA模式扫描所有可见WiFi
2. **目标筛选**: 根据配置选择特定或全部网络
3. **密码尝试**: 按字典顺序尝试每个密码
4. **连接测试**: 每个密码尝试连接15秒超时
5. **结果通知**: 成功时发送飞书通知并记录

### 安全特性

- AP模式独立运行，不影响现有网络
- 配置信息存储在本地EEPROM
- 支持WPA/WPA2加密网络破解
- 自动跳过开放网络(无密码)

### 性能优化

- 使用PROGMEM存储默认密码字典
- 异步网络扫描，不影响Web服务
- LED状态指示，便于监控运行状态
- 智能超时机制，避免长时间阻塞

## 故障排除

### 常见问题

1. **无法连接AP**
   - 检查AP名称和密码是否正确
   - 确认设备支持2.4GHz WiFi
   - 重启ESP8266设备

2. **Web界面无法访问**
   - 确认IP地址为 `192.168.4.1`
   - 检查浏览器是否支持HTTP
   - 查看串口输出确认服务状态

3. **破解速度过慢**
   - 减少密码字典数量
   - 选择信号较强的目标网络
   - 调整连接超时时间

4. **飞书通知失败**
   - 检查Webhook URL格式
   - 确认网络连接正常
   - 查看串口错误信息

### 串口调试
